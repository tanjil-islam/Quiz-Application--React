## ✨ Features

- **Interactive Quiz:** Real-time score tracking and result display
- **Infinite Scroll:** Seamless question browsing without pagination
- **Client-Side Navigation:** Smooth routing with React Router DOM
- **Firebase Backend:** Real-time data storage and retrieval
- **Performance Optimized:** Optimized production build for fast loading
- **Responsive Design:** Fully responsive across all devices
- **Clean UI:** Modern, interactive, and user-friendly interface
- **Fast Deployment:** Hosted on Netlify for reliable access

## 💻 Tech Stack

- **Frontend:** React.js 19, JavaScript (ES6+), HTML5, CSS3
- **Routing:** React Router DOM
- **Backend:** Firebase (Firestore)
- **Infinite Scroll:** React Infinite Scroll Component
- **Build Tool:** Vite
- **Hosting:** Netlify

## 🛠 Installation

1. Clone the repository
   ```bash
   git clone https://github.com/tanjil-islam/Quiz-Application--React.git
   cd Quiz-Application--React
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Set up environment variables
   ```bash
   cp .env.example .env
   ```

4. Start development server
   ```bash
   npm run dev
   ```

   Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 🔐 Environment Variables

Create a `.env` file in the root directory and add your Firebase configuration:

```env
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```
